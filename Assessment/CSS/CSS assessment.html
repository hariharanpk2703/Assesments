What is CSS?
  1. CSS stands for Cascading Style Sheets.

  2. CSS describes how HTML elements are to be displayed on screen, paper, or in other media.

  3. CSS is not neccessary CSS saves a lot of work.It can control the layout of multiple web pages all at once.

  4. External stylesheets are stored in CSS File.
CSS can be addeed to HTML documnet in 3 ways:
  1. Inline - by using the style attribute inside HTML elemnets
  2. Internal - by using a style element in the head section.
  3. External - by using a style element in the head section
  4. The most common way to add CSS, is keep the styles in external CSS files.However,in this tutorial we will use inline and internal styles,because this is easier to demonstrate, and easier for you to try it yourself.
  1. When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple elements. ID, on the other hand, applies a style to one unique element.ID is also special in that you can use a special URL to link directly to an element and it's used by JavaScript.
  2. In CSS,selectors are used to target a specific elements ona webpage. Once an element has been targeted,a style or set of styles can be applied to the element.
  3. There is a wide range of selectors available.Two of the most commonly used are class and ID.Both are target elements to which a style should be applied.
